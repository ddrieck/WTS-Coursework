Time Tracker - The Deluxe Edition

For my project I will be expanding the functionality of the site I created in ITA 343a, Time Tracker. Time Tracker is meant to be a tool to record and categorize activities done throughout the day. The end result is to allow for reporting on those activities using the collected data. Data will be entered through a webform and stored in the database.
